# 🍔 A&A Food Delivery

A modern Food Delivery Web Application developed using Angular, ASP.NET Core Web API, Entity Framework Core, and SQL Server.

---

## 📌 Features

- Home Page
- Food Menu
- Shopping Cart
- Checkout
- Order History
- PDF Invoice Download
- User Login
- User Registration
- Contact Page
- Responsive Design

---

## 🛠 Technologies Used

### Frontend
- Angular
- TypeScript
- HTML
- CSS
- Bootstrap

### Backend
- ASP.NET Core Web API
- Entity Framework Core

### Database
- SQL Server

## 🚀 How to Run

### Backend

1. Open `AAFoodDelivery.API`
2. Restore NuGet packages.
3. Run the project.
4. Swagger opens automatically.

### Frontend

```bash
cd AAFoodDelivery.Client
npm install
ng serve
```

Open:

```
http://localhost:4200
```
